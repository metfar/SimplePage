A story of a little example
The final task of the course Introduction: HTML & CSS For Beginners from Eduonix was to write a page incorporating some elemets such as:

  • Title
  • Headings
  • Paragraphs
  • Links
  • Ordered List
  • Unordered List
  • Inline attributes to structure data inside the paragraph.

The 'creative' process

I had my own agenda at this point. I wanted to try to:

  1. Create a page without using external images (just embedded svgs drawn with Inkscape);
  2. Play with gradients and transparencies as well as with rounded figures;
  3. Find new resources to use.
  4. To get acquainted with Brackets.
  5. Practice more and more HTML+CSS!!!

The conclusion

I started this page on my regular wordprocessor on which I usually work for the university tasks, the magnificent GhostWriter MarkDown Editor. It is simple to write on it but it is not an HTML Editor, but it can export to HTML. The advantage of it is that you can focus on writing and forget about the format (markDown is almost simple text).

After that, I started to add some colour to the page (I left some links in the references). And I wanted that each section will have a button to return to the Index/MainMenu. So, I made a simple arrow to go back and I made some tests. I tried to insert text on it but if it was a word, it could be resized badly, and to add the text as a form, it was so large to copy it into the text. I thought about to add it in a blob format, but I love vectorial images (sorry about that).

The last step was to clean the source and to add the proper HTML format. This was a crude process of fighting againt Brackets but I think I was able to reach the goal. At the end, it helped me to find a lot of nesting problems and to navigate my source code closing sections to do it briskly. I think I could get used to using it for a couple of years. It's not so bad!
Ah, I also tried the target field for anchor, so, the Reference section will be opened in a new tab or window, depending of which internet navigator are you using.


Thank you very much for this excelent course!!!



P.S. Sorry about the colours I choose. I promise I'll try to use a pastel palette of colours from here on.
